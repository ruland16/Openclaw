# LENS COLLABORATION PROTOCOL

## Overview
This document defines the collaboration workflow between Buzz (Chief Creative Officer - YouTube) and Lens (Media Producer) for the DJ Detective Content Pipeline project and general YouTube content production.

## Roles & Responsibilities

### Buzz (Chief Creative Officer)
- **Primary:** Content strategy, ideation, scripting
- **Secondary:** Audience analysis, performance optimization
- **Tertiary:** Monetization strategy, channel growth

### Lens (Media Producer)
- **Primary:** Technical production, editing, automation
- **Secondary:** Tool/skill implementation, quality control
- **Tertiary:** Publishing optimization, technical troubleshooting

## Communication Channels
### Primary: Shared Documents
- **Content Calendar:** `/content/calendar/CONTENT_CALENDAR.md`
- **Script Repository:** `/content/scripts/`
- **Idea Repository:** `/content/ideas/`

### Secondary: Status Updates
- **Daily Check-ins:** Brief updates via shared status document
- **Weekly Sync:** Detailed review of progress and blockers
- **Emergency:** Direct notification for urgent issues

## Workflow: Idea to Publication

### Phase 1: Ideation (Buzz)
1. **Idea Generation**
   - Create idea using `/content/ideas/IDEA_TEMPLATE.md`
   - Add to `/content/ideas/IDEAS_BACKLOG.md`
   - Tag with priority and estimated effort

2. **Concept Validation**
   - Research market/competition
   - Define target audience
   - Estimate production requirements

3. **Approval Process**
   - Submit to weekly content review
   - Get feedback from Lens on feasibility
   - Final approval by Buzz

### Phase 2: Scripting (Buzz)
1. **Script Development**
   - Use `/content/scripts/SCRIPT_TEMPLATE.md`
   - Include visual/audio requirements
   - Specify production notes

2. **Script Review**
   - Lens reviews for production feasibility
   - Technical requirements identified
   - Timeline estimation

3. **Final Script**
   - Approved script moved to production queue
   - Assets list created
   - Production schedule set

### Phase 3: Production (Lens)
1. **Pre-production**
   - Gather required assets
   - Set up production environment
   - Technical preparation

2. **Production**
   - Execute according to script
   - Quality control checks
   - Iterate based on feedback

3. **Post-production**
   - Editing and finalization
   - Optimization for platform
   - Quality assurance

### Phase 4: Publication (Buzz + Lens)
1. **Pre-publish Checklist**
   - SEO optimization (Buzz)
   - Thumbnail creation (Buzz)
   - Description writing (Buzz)
   - Technical checks (Lens)

2. **Publication**
   - Upload to YouTube (Lens)
   - Schedule publication (Buzz)
   - Initial promotion (Buzz)

3. **Post-publish**
   - Performance monitoring (Buzz)
   - Engagement management (Buzz)
   - Analytics review (Both)

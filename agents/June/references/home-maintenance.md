## Lawn & Garden Care

### Bermuda Grass Schedule
*North Carolina specific*

**Spring (March-May):**
- [ ] Apply pre-emergent herbicide (early March)
- [ ] First mow when grass reaches 2 inches
- [ ] Apply spring fertilizer (April)
- [ ] Aerate if soil is compacted (April/May)
- [ ] Begin regular mowing (weekly)

**Summer (June-August):**
- [ ] Mow 1-2 times per week
- [ ] Water deeply 1-2 times per week (early morning)
- [ ] Apply summer fertilizer (June)
- [ ] Monitor for pests (chinch bugs, armyworms)
- [ ] Apply insecticide if needed

**Fall (September-November):**
- [ ] Apply fall fertilizer (September)
- [ ] Overseed with ryegrass if desired (October)
- [ ] Reduce mowing frequency as growth slows
- [ ] Apply pre-emergent for winter weeds (November)
- [ ] Final mow at lower height (late November)

**Winter (December-February):**
- [ ] Mow only if ryegrass was overseeded
- [ ] Keep leaves off lawn
- [ ] Plan next season's care

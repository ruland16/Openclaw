## Workflow Guidelines

### Campaign Development Process
1. **Brief** → Campaign objective, target audience, budget, timeline
2. **Creative** → Copy, visuals, landing pages
3. **Execution** → Launch, monitoring, optimization
4. **Analysis** → Results, learnings, ROI calculation
5. **Archive** → Store in `memory/campaigns/` for future reference

### Store Audit Process
1. **Discovery** → Current state analysis
2. **Assessment** → Identify opportunities and issues
3. **Prioritization** → Rank by impact and effort
4. **Implementation** → Execute improvements
5. **Measurement** → Track performance changes

### A/B Testing Protocol
1. **Hypothesis** → What we expect to improve
2. **Design** → Control vs. variant
3. **Execution** → Run test with statistical significance
4. **Analysis** → Results and statistical validity
5. **Implementation** → Roll out winning variant
